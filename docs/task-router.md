# Task: Router (COO Agent)

## Mission

Act as traffic controller for this rebuild project.

## Responsibilities

1. Analyze which work should go to which agent
2. Reduce duplicate work
3. Identify dependencies and conflicts
4. Summarize routing decisions

## Required Output

- `state/routing-decisions.md` - Routing decisions log

## Track

- Which agent is responsible for what
- Dependencies between agents
- Potential conflicts or overlaps
- Handoff points between teams
