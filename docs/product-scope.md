# Product Scope - ระบบติดตามตัวชี้วัด รพ.สต.

## ภาพรวมระบบ (System Overview)

ระบบติดตามตัวชี้วัดสำหรับรพ.สต. (โรงพยาบาลส่งเสริมสุขภาพตำบล) สังกัด อบจ.อุบลราชธานี ทั้ง 13 อำเภอ

### วัตถุปุหายหลัก
- ติดตามและรายงานผลตัวชี้วัดประสิทธิภาพ (KPI) ของหน่วยบริการสุขภาพ
- แสดงข้อมูลเปรียบเทียบระหว่างหน่วยบริการและอำเภอ
- รองรับการตัดสินใจของผู้บริหารผ่าน Dashboard

### ผู้ใช้งานหลัก (Primary Users)
1. **ผู้บริหาร อบจ.** - ดูภาพรวม, เปรียบเทียบ, ตัดสินใจ
2. **เจ้าหน้าที่สาธารณสุข** - ป้อนข้อมูล, ตรวจสอบผลงาน
3. **ผู้ตรวจสอบ/ผู้วิเคราะห์** - ดูรายงาน, วิเคราะห์แนวโน้ม

## ข้อจำกัดและสมมติฐาน

### ข้อจำกัด
1. ข้อมูลปัจจุบันเป็น mock data ต้องเชื่อมต่อฐานข้อมูลจริง
2. ยังไม่มีระบบ authentication
3. ยังไม่มีการเก็บประวัติการแก้ไข

### สมมติฐาน
1. ข้อมูลอัพเดทรายไตรมาส
2. ปีงบประมาณเริ่มตุลาคม (ปีงบประมาณ 2567 = ตุลาคม 2566 - กันยายน 2567)
3. ผู้ใช้มีความคุ้นเคยกับระบบสุขภาพไทย
