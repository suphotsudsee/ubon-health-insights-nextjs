# Second Brain

## Working Definition
This vault is a personal second brain: a maintained markdown system where raw inputs are turned into structured, searchable, linked knowledge that supports decisions and reflection over time.

## Expected Inputs
- Articles and web clips
- Notes from conversations
- Health records and self-observations
- Project documents
- Research papers
- Questions whose answers are worth keeping

## Expected Outputs
- Source summaries
- Topic pages
- Decision-ready syntheses
- Durable query notes
- A growing index of what is known, uncertain, or missing

## Current Constraint
The system is only as good as the sources it captures and the discipline with which the agent updates derived pages.
